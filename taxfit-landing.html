📚 이용 가이드

CSV/Excel/이미지 파일로 세금계산서를 한 번에 불러오세요!

  • 상단의 "파일을 드래그하거나 클릭하여 업로드" 영역에 파일을 올려놓거나 클릭합니다.
  • 지원 형식: CSV, Excel(.xlsx, .xls), 이미지(JPG, PNG, GIF, WebP)
  • 한 번에 여러 세금계산서 데이터를 불러올 수 있습니다.

📋 CSV 형식 예시:

공급자명,사업자번호,공급가액,부가세,합계,발행일,계정과목 김태호,123-45-67890,450000,45000,495000,2026-07-02,용역비 박영수,234-56-78901,500000,50000,550000,2026-07-03,광고비

필수 항목 순서: 공급자명 → 사업자번호 → 공급가액 → 부가세 → 합계 → 발행일 → 계정과목

입력자 정보를 입력하세요

  • 필수 팀명: 소속 팀 선택
  • 필수 부서: 소속 부서 선택
  • 필수 담당자명: 승인 담당자의 이름
  • 필수 이메일: 담당자 이메일 주소
  • 연락처, 직책은 선택항목입니다.

세금계산서 정보를 입력하세요

  • 필수 공급자명: 서비스/물품을 제공하는 회사 또는 개인명
  • 필수 사업자번호: 123-45-67890 형식으로 자동 포맷됩니다
  • 필수 공급가액: 부가세를 제외한 금액
  • 필수 부가세: 일반적으로 공급가액의 10%
  • 필수 합계금액: 공급가액 + 부가세 (자동 검증됩니다)
  • 필수 발행일: 세금계산서 발행 날짜
  • 필수 계정과목: 용역비, 광고비, 외주비, 회의비, 기타 중 선택
  • 비고는 추가 정보 입력이 필요할 때만 기입합니다.

별표(*) 표시된 항목은 필수 입력 항목이므로 반드시 입력해야 등록됩니다.

상태 배지의 의미를 알아보세요

  • 검토대기 ⏳ 방금 등록했거나 수정 후 다시 등록한 상태
  • 승인완료 ✅ 회계담당자가 승인한 상태 (회계 처리 가능)
  • 반려됨 ❌ 회계담당자가 반려한 상태 (수정 후 재등록 필요)

필터 버튼으로 상태별 조회

  • "전체" 버튼: 모든 세금계산서 조회
  • "검토대기 ⏳" 버튼: 승인 대기 중인 항목만 조회
  • "승인완료 ✅" 버튼: 승인된 항목만 조회
  • "반려됨 ❌" 버튼: 반려된 항목만 조회

레이아웃 전환

  • ⬜ 카드 버튼: 카드 형태로 시각적으로 볼 수 있습니다
  • 📋 리스트 버튼: 목록 형태로 한 줄씩 비교할 수 있습니다

페이지네이션 사용

  • 목록이 5개를 초과하면 "이전", "다음" 버튼이 나타납니다
  • 버튼을 클릭해 다른 페이지의 항목을 조회하세요

Q. 반려된 세금계산서는 어떻게 수정하나요?

A. 반려된 항목을 클릭하면 반려 사유를 확인할 수 있습니다. 사유에 맞게 정보를 수정한 후 다시 등록하면 됩니다. 기존 항목은 자동으로 삭제되고 새로운 항목으로 등록됩니다.

Q. 소속 팀은 어떻게 변경하나요?

A. 헤더의 "소속 팀" 드롭다운에서 다른 팀을 선택하면 그 팀의 세금계산서만 조회됩니다. 새로 등록할 때는 입력 폼의 "팀명"에서 원하는 팀을 선택하고 등록하세요.

Q. 업로드 가능한 파일 형식은?

A. CSV(.csv), Excel(.xlsx, .xls), 이미지(JPG, PNG, GIF, WebP) 파일을 지원합니다. 이미지는 촬영 또는 스캔한 세금계산서 사진으로 참고용입니다.

Q. 합계금액이 자동으로 계산되나요?

A. 아니요. 직접 입력해야 하며, 공급가액 + 부가세 = 합계금액이 맞는지 자동으로 검증됩니다. 맞지 않으면 등록이 안 되므로 확인 후 다시 입력하세요.
